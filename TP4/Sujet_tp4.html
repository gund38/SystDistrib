Serveur TCP avec multiples clients en C

Le but de ce TP est de réaliser en C une application distribuée de type client/serveur avec comme support de communication les sockets TCP.

Le serveur est capable de réaliser un ensemble de services de calculs qui sont appelées à distance par les clients.

Les services de calcul que le serveur peut effectuer sont les suivants :

  • Calculer le factoriel d'un nombre n
  • Calculer la puissance x d'un nombre n
  • Calculer la moyenne d'une série de nombres et en retourner également le min et le max. La série contient un nombre quelconque de nombres.

Plusieurs clients doivent pouvoir être connectés simultanément au serveur.

Sources des fonctions de calculs : calculs.c calculs.h
